# Background
`AnnoTS` is a graphical user interface designed for annotation of time series data

# Installation
- Download the repository
- Open command prompt or anaconda prompt
- `pip install -r requirements.txt`

# Downloading Repo
- Click 'Code'
- Click 'Download Zip'

- Open command prompt or anaconda prompt
- `cd ./GUI`
- `python main.py`
