# Background
QualAI-Signal is a graphical user interface designed for annotation of time series data

# Installation
- Download the repository
- Open command prompt or anaconda prompt
- `pip install -r requirements.txt`

# Usage
- Open command prompt or anaconda prompt
- `cd ./GUI`
- `python run main.py`
